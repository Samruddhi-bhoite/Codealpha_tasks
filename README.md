# Codealpha_tasks

This repository contains the Python tasks completed as part of my internship.  
Each task focuses on core Python programming concepts such as loops, functions,
conditionals, dictionaries.

# Tasks Included

###  Task 1: Hangman Game
A simple text-based Hangman game where the player guesses a word letter by letter.

### Task 2: Stock Portfolio Tracker
A basic stock tracker that calculates total investment based on user input and
hardcoded stock prices.

### Task 3: Basic Chatbot
A simple rule-based chatbot that responds to predefined user inputs.
